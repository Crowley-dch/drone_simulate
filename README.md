# drone_simulate

## Описание
Модуль `physics.py` реализует имитационную модель движения беспилотного летательного аппарата (БпЛА) типа квадрокоптер в соответствии с пособием:

> Тельный А.В., Монахов М.Ю. Имитационная модель движения беспилотного летательного аппарата типа квадрокоптер. Системы управления, связи и безопасности, 2025. № 4.
## Реализованные формулы

| Формула | Описание |
|---------|----------|
| (1) | Система дифференциальных уравнений движения (12 уравнений) |
| (2) | Коэффициенты a1...a8 |
| (3) | Перевод напряжения в угловую скорость винтов |

### Структура модуля

### Класс DroneParams
Хранит параметры дрона. Значения по умолчанию соответствуют Таблице 1 пособия.

|Параметр |	Описание  |	Значение по умолчанию|
|---------|-----------|----------------------|
|m_c  |	масса корпуса  |	0.692 кг|
|m_r  |	масса винта  |	0.008 кг|
|m_l  |	масса луча с двигателем  |	0.094 кг|
|l  |	длина луча  |	0.25 м|
|k  |	коэффициент тяги  |	3e-6|
|b  |	коэффициент сопротивления  |	1e-7|
|KV |	оборотов на вольт в минуту  |	1000|
|V_MIN  |	минимальное напряжение  |	0 В|
|V_MAX  |	максимальное напряжение  |	12 В|
|g  |	ускорение свободного падения  |	9.81 м/с²|

### Класс QuadcopterDynamics
Основной класс, реализующий динамику полета.
