# drone_simulate

## Описание
Модуль `physics.py` реализует имитационную модель движения беспилотного летательного аппарата (БпЛА) типа квадрокоптер в соответствии с пособием:

> Тельный А.В., Монахов М.Ю. Имитационная модель движения беспилотного летательного аппарата типа квадрокоптер. Системы управления, связи и безопасности, 2025. № 4.
## Реализованные формулы

| Формула | Описание |
|---------|----------|
| (1) | Система дифференциальных уравнений движения (12 уравнений) |
| (2) | Коэффициенты a1...a8 |
| (3) | Перевод напряжения в угловую скорость винтов |
